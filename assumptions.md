# Assumptions

1. All ID columns are globally unique.

2. Student emails and roll numbers are unique.

3. One student cannot enroll in the same course multiple times simultaneously.

4. One attendance record exists per student per session.

5. Problems can exist independently of contests.

6. Contest-problem relationships are many-to-many.

7. Runtime and memory values cannot be negative.

8. Difficulty values are limited to Easy, Medium, and Hard.

9. Raw import tables may contain inconsistent or duplicate data.

10. Data cleaning should occur before insertion into normalized tables.
